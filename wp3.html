Overview

WP3 develops the core technical infrastructure of the Enact platform — a sophisticated, interoperable repository capable of capturing the full complexity of practice research outputs, from single digital objects to complex multi-modal portfolios.

Built in close collaboration with Pathfinder projects and the Community Advisory Group, the platform is designed around real researcher needs rather than assumed ones.

Three Repository Layers

Dissemination Layer

Public-facing interface for browsing, searching, and engaging with practice research outputs. Designed for researchers, institutions, funders, and the public.

Functionality Layer

Deposit workflows, metadata editing, rights management, embargo controls, and collaborative features for researchers and institutional administrators.

Preservation Layer

Integration with digital preservation systems, archival format conversion, and fixity checking to ensure long-term accessibility of deposited outputs.

Key Technical Features

  • Support for diverse media types: video, audio, image sequences, 3D models, documents, datasets, and mixed-media portfolios
  • Integration with ORCID, DOI, RAiD, and CRediT persistent identifier systems
  • Portfolio and project-level aggregation of related outputs
  • Flexible access controls and embargo functionality for sensitive or commercially sensitive work
  • API access for institutional repository integration and harvesting
  • FAIR-compliant metadata and discovery interfaces
